# SV Engine
SV Engine is simple server engine.

## Features
### Windows API
It used [NetCpp](https://github.com/index1207/netcpp) library that based on IOCP.

### Logging
Through `Console` class to log server status.

### Generated Packet
Write packet structure with json format on `packet.json` and run `generate.py` to generate packet source file. <br>
It will support many languages.

#### support languages
- [x] C++
- [ ] C#
